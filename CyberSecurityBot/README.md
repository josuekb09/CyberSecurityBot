# Cybersecurity Awareness Bot

**Student Name:** Kabuya Tambwe Josue
**Student Number:** ST10468057

A console-based chatbot developed in C# that educates users on online safety, including password safety, phishing, and safe browsing.

## How to Run the Application
1. Clone the repository to your local machine.
2. Open the solution in Visual Studio.
3. Press `F5` or click the "Start" button to compile and run. 
4. Ensure your volume is up to hear the voice greeting!
